Implementing state structures to control user interactivity, enabling real-time capture and mathematical calculation variables across text input components.

Applying customized Material Design 3 guidelines to fit clean-energy themes, customizing color palettes, elevation states, and clear text hierarchies.

Building complete structural app navigation using Compose Jetpack Navigation controllers and separating state objects using ViewModels.
This session showed me how to structure multi-screen applications. Utilizing a central ViewModel allowed persistent transfer of calculations and structural configurations safely between different navigation destination views.
